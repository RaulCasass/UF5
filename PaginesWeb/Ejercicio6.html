LISTA CON DEFINICIONES

HTTP
- Es un protocolo de comunicación utilizado para la transmisión de información en la World Wide Web.
HTML
- Es el lenguaje estándar utilizado para la creación y el diseño de páginas web.
URL
- Es una dirección única que se utiliza para identificar recursos en la web.
TCP/IP
- Es el conjunto de protocolos que rige la transmisión de datos en la Internet.
Internet
- Es una red global de computadoras interconectadas que utilizan el conjunto de protocolos TCP/IP para compartir información y recursos.
W3C
- Es una organización internacional que desarrolla estándares para la World Wide Web con el objetivo de garantizar su crecimiento a largo plazo.
Mozilla Foundation
- Es una organización sin fines de lucro que desarrolla software de código abierto.
CSS
- Es un lenguaje de diseño utilizado para controlar la presentación y el formato de un documento HTML.
